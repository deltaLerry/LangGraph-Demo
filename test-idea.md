# 项目名    
我靠规则在怪谈封神

# 类型方向
规则怪谈
单个副本围绕一个核心场景展开，规则和谜题集中。可结合《十日终焉》的规则智斗与《道诡异仙》的氛围营造，但将规模控制在一个副本内。
多个副本需要有串联的线 明线暗线等技巧。副本转场可以参考《蛊真人》等经典转场。

# 核心看点与爽点
在既定规则下智斗求生、探索真相的悬疑感和解谜快感。
副本成长，一步步变强，突破规则。

# 注意
每个副本10~20章节的内容。副本之间不要有割裂感。自己根据章节数量规划副本的个数。
每个章节的情节需要完整。每个章节都需要有标题。

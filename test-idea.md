# 项目名    
规则怪谈：无限

标签
无限流、规则怪谈、副本升级流

# 参考
（并不限于这个，不能抄袭，要原创，当然可以借鉴）
* 规则智斗：《十日终焉》
* 氛围营造：《道诡异仙》
* 副本地图转场：《蛊真人》
* 无限流可以参考：《无限恐怖》
* 副本升级流可以参考：《凡人修仙传》


# 核心看点与爽点
在既定规则下智斗求生、探索真相的悬疑感和解谜快感。
有副本成长，一步步变强，突破规则。副本转场经典流畅。
多个副本需要有串联的线 明线、暗线等。


# 注意
每个副本10~20章节的内容。副本之间不要有割裂感。自己根据章节数量规划副本的个数。
每个章节的情节需要完整。每个章节都需要有标题。


文风：
第三人称/第一人称（按项目选择）
- 节奏：短句为主，关键场景拉长描写
- 禁止：AI味总结句、机械重复句式、无意义的套话、拒绝ai风味，增强活人感。
- 鼓励：偶尔幽默来减低紧张感（但不要影响主风格）
- 鼓励：文笔优美


点子：
普通人被卷入无限的规则怪谈，在看似无关的副本中挣扎求生，一步步成长，逐渐接近了这个世界的真相...
